# Calculator_on_CPP
Это калькулятор, основанный на инструкции при установке Visual Studio

Это программа знакомит с:
- структурой классов
- ведением проектов
- Сборка файлов

Версия 1.0
- Представлены базовые вычисления для дувух чисел.
- Учтена проверка деления на ноль.


## Нынешние баги и будущие исправления
- при введении буквы вместо числа происходит зацикливание вывода ошибки
- нужно добавить графический интерфес
- добавить больше функций калькулятора
- для уменьшения возможных ошибок ввода можно ввести управление через интерфейс

*Возможно  все эти исправления будут добавлены в версии два этого калькулятора. Тут будет пока рассмотрен мозг и ведение  проекта.*
